PUZ-PRE v3 Editor's Link

PUZ-PRE (Tentative) is a Web Application that is for editing and playing the puzzles published by nikoli. I developed the script based on Java Applet at first, and I develop the JavaScript version of the script now.

I develop the script mainly with Firefox 3.5.5 and Safari 4.0.4, test on Google Chrome 3.0, Internet Explorer 6/8 and Sometimes firefox 3.0 and Opera 9.64/10.

But due to InternetExplorer or another IE component browser's processing speed, I recommand another browsers to use PUZ-PRE v3. PUZ-PRE v3's drawing process on IE uses the library uuCanvas.js v1.0. If you install Silverlight, drawing processing speed on IE is faster by several times than not-installed.
