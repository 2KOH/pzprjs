    About PUZ-PRE v3

    PUZ-PRE(Tentative) is a Web application that is for editing and playing the puzzles that is mainly published by nikoli. I develop the script by JavaScript, and also work for implementing HTML5 feature.
    I develop the script mainly with Firefox 3.6 and Opera 10.51, test on Google Chrome 4.0, Safari 4.0 and Sometimes Internet Explorer 6/8, Firefox 3.0+ and Opera 9.64+.
    to Japanese Menu

    Browsers recommanded

    Firefox 3.0+, Google Chrome, Safari 4.0+, Opera 10.50+
    - Because Opera 10.51 is unstable for condtional branch, I remove from them currently.

    Browser requirements

    Firefox 2.0+, Google Chrome, Safari 3.2+, Opera 9.64+, Internet Explorer 6.0 or 8.0,
    or another browsers that uses IE component (Trident). But due to Internet Explorer or another IE component browser's processing speed, I recommand another browser to enjoy PUZ-PRE v3.
