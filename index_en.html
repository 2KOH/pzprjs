    About PUZ-PRE v3

    PUZ-PRE is a Web application that is for editing and playing the puzzles that is mainly published by nikoli. I develop the script by JavaScript, and also work for implementing HTML5 feature.
    I develop the script mainly with Firefox 3.6, test on Google Chrome 5.0, Safari 4.0 and Sometimes Internet Explorer 8.0, Firefox 3.0+ and Opera 9.64+.
    to Japanese Menu

    Browsers recommanded

    Firefox 3.5+, Google Chrome 4.0+, Safari 4.0+, Opera 10.60+, Internet Explorer 9.0

    Browser requirements

    Firefox 2.0+, Google Chrome, Safari 3.2+, Opera 9.64 to 10.10, 10.60+ or Internet Explorer 8.0+, or another browsers that uses IE component (Trident). PUZ-PRE v3 supports iOS4 limitedly.

    Restriction

    • The temporary save function doesn't work with Firefox1.5, Google Chrome3.0, Safari3.2, Opera10.10 or earlier browsers.
    • The output-image function works incorrectly with Firefox3.0, Google Chrome1.0, Safari3.2, Opera10.10, InternetExplorer8.0 or earlier browsers.
    • Google Chrome 6.0 is not supported for its unstableness of SVG and CSS.
    • Opera 10.5x is not supported for its unstableness of condtional branch.
    • Due to Internet Explorer or another IE component browser's performance, I recommand another browser to enjoy PUZ-PRE v3.
